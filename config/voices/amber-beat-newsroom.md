# Amber Beat — Weekly Global Student Accommodation Market News

## What It Is

A weekly snapshot of key developments in top student accommodation markets. Published at amberstudent.com/news/amberbeat every week.

**Tagline:** "Missed keeping track of all the latest happenings in the student accommodation market? Amber Beat is a weekly snapshot of the key developments in top student accommodation markets."

## Structure

1. **Header**: "Amber Beat | News from [date range]"
2. **Intro**: Brief tagline about Amber Beat
3. **Regions in fixed order**:
   - United Kingdom
   - United States of America
   - Australia
   - Canada
   - Europe
4. Each region has **5-9 news items**
5. Each item ends with **[Source](URL)** link

---

## Item Format (STRICT)

Each item is a **single sentence, 20-30 words**, neutral tone, fact-led. Must include at least one specific data point where available.

## Topic Priority (within each region)

1. **Policy & regulatory shifts** — study visas, post-study work (PSW/OPT/PGWP), enrolment caps, immigration/compliance
2. **International student enrolment / demand** — application volumes, visa issuance, YoY enrolments, source-market mix
3. **University-related** — rankings, institution enrolment, funding, campus/program changes
4. **PBSA supply & investment** — M&A, asset sales, new developments, construction, bed growth

---

## Style Rules

- **Neutral tone** — factual, not opinionated
- **Data-led** — include numbers, percentages, monetary figures wherever possible
- **One sentence per item** — concise, scannable
- **Source link at end** — [Source](url) format
- **No promotional language** — no "amber thinks", no branding
- **No opinion pieces** — only factual news reports and data releases
- **Region order fixed** — UK → USA → Australia → Canada → Europe
